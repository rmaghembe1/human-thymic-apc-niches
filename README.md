# Human thymic stromal-immune antigen-presenting niches

This repository contains reproducibility materials for the manuscript:

Single-cell and spatial reconstruction of human thymic stromal-immune antigen-presenting niches across early-life development

Author: Reuben S. Maghembe

## Study summary

This study re-analyses public human thymus single-cell, CITE-seq and Visium spatial transcriptomic datasets to reconstruct thymic epithelial, stromal and immune antigen-presenting niche programmes.

The analysis uses Bautista human thymic stromal single-cell data as a reference for thymic epithelial and stromal architecture, and Heimli paediatric thymus CITE-seq and Visium data to resolve immune antigen-presenting, thymic B-cell, epithelial, stromal and spatial niche programmes.

## Repository structure

- scripts: reproducibility scripts.
- tables: processed source tables and annotation tables.
- figures: main and supplementary figure files.
- docs: documentation, checklists and reference tables.
- metadata: reserved for metadata files.
