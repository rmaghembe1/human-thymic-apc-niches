# Data availability

All primary datasets analysed in this study are publicly available from the Gene Expression Omnibus.

- GSE147520: Bautista human thymic stromal single-cell dataset.
- GSE207206: Heimli paediatric thymus CITE-seq and Visium spatial transcriptomic dataset.

This repository provides processed source tables, annotation files, figure outputs and analysis scripts generated during the re-analysis. Raw sequencing data and raw GEO archives should be retrieved directly from GEO.
